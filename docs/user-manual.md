# Manuel utilisateur

## Connexion
- Ouvrir l'application
- Entrer identifiants
- Valider l'acces securise

## Dashboard
- Vue globale (agents, equipements, incidents)
- Indicateurs rapides et priorites

## Agents
- Ajouter un agent
- Mettre a jour ses informations
- Suivre le statut actif/inactif

## Inscription agent
- L'agent clique sur "Inscription agent" depuis l'ecran de connexion
- Il renseigne ses informations personnelles et projet
- Le compte est cree automatiquement avec le role Agent
- Informations obligatoires: nom, prenom, telephone, numero piece, photo piece, type de projet

## Equipements
- Enregistrer un equipement (type, numero, IMEI)
- Consulter l'etat et le statut
- QR code genere automatiquement

## Affectations
- Creer une affectation (equipement + agent)
- Suivre la date et le statut
- Telecharger la fiche PDF

## Restitutions
- Enregistrer la restitution
- Noter l'etat et les observations
- Mise a jour automatique du statut

## Incidents
- Declarer un incident (perte, vol, panne)
- Suivre le statut jusqu'a cloture

## Rapports
- Consulter les indicateurs
- Exporter en Excel ou PDF
